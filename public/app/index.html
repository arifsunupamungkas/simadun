Manajemen Arsip Surat Masuk/Keluar SPT Dinas Cetak PDF

Dashboard Rekapitulasi

Pantau seluruh statistik dokumen secara real-time. Total Arsip, Surat Masuk, Keluar, Undangan, dan SPT tersaji dalam kartu yang mudah dibaca.

Upload Dokumen Pintar

Unggah file PDF/gambar dengan seret-dan-lepas. File diproses otomatis via AppScript Bridge untuk mengoptimalkan kuota Google Drive Anda.

Edit Data Universal

Setiap data yang tersimpan dapat diedit kapan saja, termasuk mengganti lampiran file. File lama di Drive otomatis tergantikan oleh file baru.

Cetak Laporan Berkop

Ekspor rekap data menjadi PDF resmi berkop surat Inspektorat lengkap dengan logo. Kolom Aksi otomatis tersembunyi agar cetakan tampil rapi.

1
Memulai: Login & Navigasi Utama
Halaman Login

Akses SIMADUN melalui URL yang diberikan. Masukkan Username dan Password yang telah disiapkan oleh Administrator. Klik "Masuk ke Sistem".

Navigasi Sidebar

Menu navigasi berada di panel kiri (Sidebar). Klik menu untuk berpindah halaman. Di perangkat mobile, tekan ikon hamburger (≡) di pojok kiri atas untuk membuka sidebar.

Toggle Dark Mode

Klik ikon Bulan/Matahari (🌙/☀️) di pojok kanan atas untuk mengganti tema aplikasi. Preferensi tema otomatis tersimpan untuk sesi berikutnya.

Refresh Data

Gunakan tombol Refresh di halaman Dashboard atau masing-masing module untuk mengambil ulang data terbaru dari server Google Sheets.

2
Manajemen Arsip Dokumen
Upload Arsip Baru

Masuk ke menu Arsip → klik "Upload Arsip" (pojok kanan atas). Isi Nama File, Kategori, Deskripsi, dan Tanggal Arsip Asli. Seret file softcopy (PDF/Scan) ke kotak bergaris putus-putus, lalu klik Simpan.

Mencari Data Arsip

Gunakan kotak pencarian di atas tabel. Ketik kata kunci (nama file, kategori, dll) untuk menyaring data secara real-time tanpa perlu reload halaman.

Preview & Download

Klik ikon Mata (👁) di kolom Aksi untuk menampilkan preview dokumen dalam jendela popup. Klik ikon Unduh untuk langsung membuka link Google Drive.

Edit Data Arsip

Klik ikon Pena Kuning (✏️) di kolom Aksi. Modal Edit akan terbuka berisi semua data arsip yang dapat diubah. Jika ingin mengganti file, unggah file baru di bagian bawah modal — file lama di Google Drive otomatis terhapus.

Cetak Laporan Arsip

Klik "Cetak Laporan" di pojok kanan atas halaman Arsip. PDF akan dibuka di tab baru dengan format berkop surat resmi Inspektorat.

3
Manajemen Surat (Masuk, Keluar & Undangan)
Tab Navigasi

Halaman Surat memiliki 3 tab: Surat Masuk, Surat Keluar, dan Undangan. Klik tab yang sesuai untuk beralih antar kategori surat.

Input Surat Baru

Klik tombol "Tambah Surat Masuk / Keluar / Undangan" yang tersedia di masing-masing tab. Panel formulir akan muncul dari atas. Isi semua data wajib (*) lalu simpan.

Lampiran Surat

Setiap surat dapat disertai lampiran softcopy (PDF/gambar). Upload lampiran di bagian bawah formulir. Link lampiran otomatis tersimpan di database dan dapat dibuka dari tabel.

Edit & Hapus Surat

Gunakan tombol Edit (pena kuning) untuk memperbarui data. Tombol Hapus (merah) akan menghapus data secara permanen beserta filenya di Google Drive.

4
Surat Perintah Tugas (SPT)
Mode Arsip SPT

Jika SPT telah dibuat di luar sistem (MS Word, dll), gunakan tab Arsip SPT untuk mengunggah file PDF-nya. Data SPT akan tercatat di sistem lengkap dengan link dokumen aslinya.

Mode Buat SPT

Tab Buat SPT menyediakan generator dokumen resmi. Isi semua field formulir: Nomor SPT, Nama & NIP Petugas, Jabatan, Tujuan, Keperluan, dan rentang tanggal penugasan.

Cetak & Simpan SPT

Setelah formulir terisi, klik "Cetak & Simpan SPT". Dokumen SPT resmi berformat A4 dengan Kop Surat Inspektorat akan muncul di jendela baru siap dicetak. Data SPT juga otomatis tersimpan ke database.

Kustomisasi Kop Surat

Format Kop Surat (nama pemerintah, instansi, alamat, pejabat penandatangan) dapat dipersonalisasi melalui Menu Pengaturan → tab Format Cetak SPT.

5
Pengaturan Sistem & Manajemen Akun
Kelola Pengguna

Admin dapat menambahkan akun pengguna baru dan mengelola daftar pengguna melalui tab "Kelola Pengguna" di Pengaturan.

Ganti Password

Setiap pengguna dapat mengganti password-nya sendiri melalui panel "Ganti Password" yang tersedia di tab Kelola Pengguna.

Format Cetak SPT

Atur Kop Surat resmi (nama pemerintah, instansi, alamat) dan data penandatangan (kota, jabatan, nama pejabat, NIP) di tab Format Cetak SPT.

Inisialisasi Database

Jika ada Sheet yang hilang atau Database perlu di-reset, gunakan tombol "Inisialisasi Database" di tab Database. Gunakan dengan hati-hati!

Zona Bahaya: Inisialisasi Database

Tombol di bawah ini akan membangun ulang struktur Tab (Sheet) pada Google Spreadsheet yang digunakan sebagai database SIMADUN. Gunakan hanya jika ada Sheet yang hilang atau database perlu di-reset.

Proses ini tidak menghapus data yang sudah ada. Ia hanya memastikan semua sheet yang diperlukan (Arsip, Surat Masuk, Surat Keluar, Undangan, SPT, Users) tersedia.
Pastikan variabel lingkungan GOOGLE_SHEET_ID dan GOOGLE_SERVICE_ACCOUNT_KEY sudah terkonfigurasi dengan benar di Vercel sebelum menjalankan ini.
